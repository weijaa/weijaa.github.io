如何用 C# 撰寫 wox plugin

  • step 1

    • 新 .NET framework 專案
    • 安裝 nuget 套件

  • step 2

    • 將 class 繼承 IPlugin 並實作功能
    • 實作時先想好
      • 想解決甚麼問題
      • input and output

  • step 3

    • 做好後 build 一下專案
    • 到 bin -> debug 複製出所有 .dll檔案
    • 開新資料夾存著

  • step 4

    • 創建新檔案 plugin.json
    • 將 dll 檔和 json 檔壓縮成 zip檔
    • 把 zip附檔名改成 wox
    • 將 .wox 檔案拉近 wox
    • 完工!!!
